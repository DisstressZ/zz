  Вы открыли программу "Автоматизированная система составления расписания"

  С помощью этой программы вы можете составить расписание учебного заведения.
  
  Основную часть интерфейса занимает таблица. В этой таблице будет отображаться расписание выбранной группы/преподавателя. 

  Для добавления группы необходимо перейти в меню "Группы - Добавить новую" и заполнить поля. (После создания группы учебный план поменять НЕЛЬЗЯ).

  Обратите внимание на формат файла с учебным планом, который приведен в качестве образца (если хотите добавить новый учебный план): текстовый документ, первая строка выполняет роль заголовка таблицы, разделитель между полями ";". Это поможет вам легко создать и добавить свой учебный план.

  Для добавления занятия нужно выбрать группу, неделю и щелкнуть на ячейку в таблице. Затем необходимо выбрать тип занятия, дисциплину, ввести номер аудитории и нажать кнопку "Добавить занятие".

  При добавлении занятие автоматически добавляется в расписание преподавателя.

  Список дисциплин, предлагаемых для добавления, формируется на основе загруженного при добавлении группы учебного плана.

  При добавлении занятия производится автоматическая проверка занятости аудитории и преподавателя в это время. В случае если у одной группы с потока была добавлена лекция, на нее можно добавить группу, которая указана в учебном плане в списке групп на данной дисциплине.

  При удалении занятия, оно будет автоматически удалено из расписания преподавателей и групп, которые тоже присутствуют на этом занятии.

  Для проверки соответсвия расписания выбранной группы учебному плану по количеству часов необходимо нажать кнопу "Проверить группу". На экран будет выведено сообщение с результатами проверки.
